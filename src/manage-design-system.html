系统设计

系统的复杂性：

系统的数学表达：
Sys = {C,R,Stru,Env}
Sys
表示系统
C
表示组件的集合
R
表示组件之间各种关系的集合
Stru
表示组建之间的组织结构
Env
表示和环境之间的交互
系统的复杂性，F＝rc，当Stru为1个分区的时候：
Sys = {C,R,Stru,Env}F=rc
F
表示系统复杂性
c
表示组件的个数
r
表示平均每个组件与其它组件的状态关系数
